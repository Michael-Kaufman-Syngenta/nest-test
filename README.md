## Getting Started

Use `npm run start:dev` to start developing in watch mode.

## Generating new CRUD endpoint

```bash
$ nest g resource [name]
```

## Generating new controller

```bash
$ nest g controller [name]
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
